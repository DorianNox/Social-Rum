## 💡 Идея

Рум — это пространство, где люди общаются анонимно под своим уникальным эмодзи.
Ты выбираешь эмодзи один раз при регистрации — и это навсегда.
Каждое эмодзи автоматически создаёт клуб, где собираются все люди с таким же аватаром.

Хочешь общаться только с 🦊? Добро пожаловать в Клуб 🦊.

---

## ✨ Возможности

### 👤 Профиль
- Анонимная регистрация — только логин и пароль, никаких имён
- Эмодзи-аватар выбирается при регистрации и не меняется
- Обложка профиля — загрузи фото или выбери цвет
- Раздел «О себе», интересы, ссылки
- Настройки приватности постов (все / друзья / только я)

### 📰 Лента
- Две вкладки — все посты и только от друзей
- Бесконечный скролл
- Создание постов с фото
- Лайки с анимацией ❤️
- Комментарии
- Репосты

### 🏠 Клубы
- Создаются автоматически по эмодзи
- Стена клуба — посты только для участников
- Рейтинг топ-10 активных участников
- Чат клуба в реальном времени

### 💬 Общение
- Личные сообщения с реалтайм-обновлением
- Уведомления о лайках, комментариях, подписках

### 🔍 Поиск
- По пользователям и постам

### ⚙️ Безопасность
- Пароли хешируются через `werkzeug.security`
- Жалобы на посты
- Админ-панель — управление пользователями и жалобами
- Администратор полностью невидим для пользователей

### 🎨 Дизайн
- Тёмная и светлая тема
- Минималистичный стиль в духе VK / Instagram
- Адаптивная вёрстка под мобильные

## 🚀 Быстрый старт
```bash
# 1. Клонируй репозиторий
git clone https://github.com/DorianNox/rum.git
cd rum

# 2. Установи зависимости
pip install flask werkzeug

# 3. Запусти
python app.py

# 4. Открой браузер
http://localhost:5000
```
